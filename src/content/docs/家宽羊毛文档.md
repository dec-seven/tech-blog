注册 MostLogin 后会赠送若干 GB 的家宽代理流量。进入赠送页面，选择对应的国家或地区；配置好代理后，后续操作与其他 Team 一样，正常付款即可。

![MostLogin 免费代理流量赠送页面](/images/home-broadband/mostlogin-free-proxy.png)

> 如果通过上述方式无法领取或使用流量，大概率是触发了风控。可以等待几个小时后重试，也可以直接注册 IP DEEP。关注其公众号或 Telegram，可领取 500 MB 体验流量。

熟悉代理提取和浏览器配置的读者，可以跳过后续内容。

## 操作步骤

### 1. 在 MostLogin 中进入 IP DEEP

在 MostLogin 的代理赠送页面点击 **IP DEEP**，进入 IP DEEP 后，选择并提取对应国家或地区的代理：

1. 进入“动态住宅代理”。
2. 打开“提取动态住宅 IP”。
3. 选择目标国家或地区。
4. 根据需要选择协议、IP 时长和提取数量。
5. 点击“生成代理信息”。

![在 IP DEEP 中提取对应国家或地区的代理](/images/home-broadband/ipdeep-extract-proxy.png)

### 2. 在浏览器中配置代理

将提取到的代理信息配置到对应的浏览器环境中。这里以 MostLogin 为例。

### 3. 验证代理 IP

启动指纹浏览器后，访问 [IPPure](https://ippure.com/)，检查当前 IP 地址、国家或地区是否与所配置的代理一致。确认无误后，再继续后续操作。

## 提取 checkout 长链

关于 checkout 长链的提取，可以参考文末资料。核心流程如下：

1. 在进入账单页面之前，按 `F12` 打开浏览器开发者工具。
2. 切换到“网络（Network）”面板，并保持面板开启。
3. 进入账单页面，在请求列表中搜索 `checkout`。
4. 找到对应请求后，将请求复制出来。
5. 仅按实际需求修改明确了解的参数，然后在控制台中运行。

> 此操作仅用于本人账户的正常调试。不要修改金额、商品、账户归属或其他支付关键字段，也不要使用来源不明的控制台代码，以免造成账号、支付或数据安全风险。
